# Workflows

这里存放项目级自动化工作流。和 `skills/` 的区别：

- `skills/` 更像可复用能力包，入口是 `SKILL.md`。
- `workflows/` 更像具体项目或业务链路，入口通常是 `README.md`、`package.json` 或脚本。

当前工作流：

- `shopee-adspower-scraper/`：Shopee Seller Center / AdsPower 采集、Product Impact、广告小时数据等。
- `google-sheets-analysis/`：Google Sheets 数据源探测与分析辅助。

不要提交原始下载、真实凭证、临时导出和大体积文件。
