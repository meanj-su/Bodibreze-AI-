# 共享资源（公开版）

本文件是 GitHub 可同步的脱敏共享资源说明。真实本地路径、凭证路径、Spreadsheet ID、token/env 文件路径等只保留在 `E:\AI项目汇总\公共docs\共享资源.md` 或本地环境变量中。

## 数据源类型

- Google Sheets 汇总表：通过本地配置和凭证访问。
- Shopee 官方 Excel：通过 `shopee-official-excel-export` Skill 导出。
- AdsPower/CDP 页面数据：通过 `shopee-adspower-scraper` workflow 或相关 Skill 读取。
- Shopee 图表 sourceData：通过 `shopee-hover-data` Skill 读取。
- BODIBREZE SKU / Bundle：公开规则和脱敏样例放 `docs\brand\BODIBREZE`，真实源文件和工作表保留本地。

## 安全规则

- 不提交 credentials、token、cookies、private key、service account JSON 或 env 文件。
- GitHub 只保留变量名、流程说明、字段口径和脱敏样例。
- 真实数据源位置、账号范围、凭证路径只在本机安全文档或环境变量中维护。

## 可复用口径

- SKU 主名称优先使用标准 SKU 对照表，不用链接标题作为主名称。
- 分析展示优先使用中文产品名，SKU 作为辅助标识。
- 价格、补贴、discount 等口径必须在报告中说明来源字段和计算方式。
- 原始数据保持不变，清洗和派生结果写入 `Bodibreze-Data-Library`。
